# California School Explorer v0.4.2

Release date: July 15, 2026

## Family-facing changes

- Compare a school trend with its shared district, shared county, or California.
- See whether each selected school has a current value, only an older value, or no public value for every published measure.
- Copy a comparison URL that restores selected schools, the student group, metric, time range, reference context, and experimental composite weights.
- Move directly from Area Explorer results to the selected-school comparison.

## Data changes

The public bundle adds 58 county reference files and one statewide file, totaling 26,765 reference observations. Matching official CDE county aggregates are used for A–G completion, chronic absenteeism, college-going, dropout, graduation, and suspension. County ELA, mathematics, and College/Career Indicator references are calculated from public district rows using the published denominator. Suppressed district values remain excluded and are never reconstructed.

Every reference file includes metric-level basis metadata. The interface identifies official county, official statewide, and denominator-weighted district calculations separately.

## Deployment

The release remains a static Cloudflare Worker Assets deployment. The new reference bundle adds less than 1 MB and requires no D1 database, R2 bucket, model, secret, or runtime database connection.
