# California School Explorer v0.1.1

This hotfix makes the v0.1.0 real-data application compatible with Cloudflare's temporary Worker deployment limits.

## Fixed

- Split the 9,946-school statewide search index into four manifest-declared files.
- Kept every static asset below 5 MiB without removing school profile fields or statewide search coverage.
- Loaded the four index files in parallel after the versioned manifest is validated.

All indicators, subgroups, school profiles, district baselines, observations, suppression rules, and source metadata are unchanged from v0.1.0.
