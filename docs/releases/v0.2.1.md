# California School Explorer v0.2.1

This patch release makes public-data caching safe across deployments.

School-index, school-shard, and district-shard requests now include the current release identifier in the URL. Returning browsers therefore fetch the observation bundle that matches the newly revalidated manifest instead of reusing an older release under the same asset path.

Data content, indicators, source snapshots, suppression rules, and school coverage are unchanged from v0.2.0.
