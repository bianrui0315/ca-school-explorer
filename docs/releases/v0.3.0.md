# California School Explorer v0.3.0

This release adds the third adjacent year of real CDE outcomes and introduces a broader visual comparison workspace.

## Data

- Five checksum-pinned 2022–23 CDE snapshots add 1,250,708 canonical facts.
- The canonical PostgreSQL store now contains 3,775,696 facts across 15 outcome snapshots plus one current school-profile snapshot.
- The public Worker bundle contains 2,739,483 observations for 9,946 schools and 1,023 district baselines.
- ELA, mathematics, chronic absenteeism, suspension, graduation, A–G completion, and dropout now cover 2022–23 through 2024–25.

## Experience

- Trend endpoints are inset from chart boundaries and show three annual points by default.
- The seven-axis profile maps every available latest-year indicator to a documented 0–100 comparison scale.
- The experimental composite exposes editable weights, available-indicator count, and available-weight coverage. It is explicitly not an official rating, percentile, or ranking.
- Missing and inapplicable indicators remain absent; they are not plotted or scored as zero.
- The selected-school map uses quality-controlled published coordinates, loads Leaflet on demand, and includes visible OpenStreetMap attribution and attendance-boundary cautions.

## Deployment

The site remains a static Cloudflare Worker deployment. PostgreSQL is only used to generate release assets. The map adds no Worker database, API, secret, or server-side compute requirement.

Run `npm run release:check` before deployment, then `npm run release:deploy` after Cloudflare authentication.
