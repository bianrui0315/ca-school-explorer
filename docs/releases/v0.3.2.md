# California School Explorer v0.3.2

This release makes the location evidence finder personalizable and reproducible without turning it into a universal school rating.

## Experience

- Select an exact child grade or keep the elementary, middle, and high school overview.
- Limit results to district-operated, charter, or magnet public schools.
- Require 50%, 70%, 90%, or 100% of the selected evidence weight.
- Give academic performance, attendance, school climate, and high-school readiness less, standard, more, or highest priority.
- See the strongest evidence driver for every eligible school match.
- Copy a validated link that restores the displayed search center, radius, filters, evidence threshold, and priorities.

## Method guardrails

- Priorities multiply documented grade-band weights and never change official source values.
- Every eligible result still requires reliable academic evidence.
- Distance remains outside the evidence score and only breaks ties.
- Protected characteristics do not steer housing-location eligibility or ordering. Official subgroup data remains available after a school is added to comparison.
- Nearby does not mean assigned or eligible, and the evidence score is not an official CDE rating.

## Deployment

The feature uses the existing static public read model and adds no database, model, key, or Worker-side state. Run `npm run release:check`, deploy with `npm run release:deploy`, and verify a shared location URL after publication.
