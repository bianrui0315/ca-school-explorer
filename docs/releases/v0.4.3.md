# California School Explorer v0.4.3

Release date: July 15, 2026

## Family-facing changes

- Choose any selected school as an anchor and find six public schools with a similar institutional profile.
- See plain-language reasons for each match, including grade-span, enrollment, and public student-profile similarities.
- Add a matched school directly to the existing comparison.
- Use the six matched schools as a reference line in the trend chart and exact-value table.
- Share a comparison URL that restores the peer reference and anchor school.

## Matching method

Candidates must have the same school level, a compatible grade span, the same DASS status, and a compatible virtual-school setting. The remaining distance uses enrollment, school type and designations, and the latest public percentages for English learners, students with disabilities, and socioeconomically disadvantaged students.

Academic performance, attendance, suspension, graduation, dropout, College/Career Indicator, college-going, and the experimental composite are never used to select or order peers. The interface publishes match reasons but no numeric similarity score, quality score, percentile, or rank.

The peer reference is calculated only after matching. For each metric, student group, and year, usable public school rows are weighted by their published student denominator. Suppressed and unavailable rows are excluded and never reconstructed.

## Data changes

The browser-safe public school index now includes three profile percentages for all 9,946 published public schools. Outcome observations remain in separate school shards and are loaded only for the six matched peers needed to calculate a selected reference.

## Deployment

The release remains a static Cloudflare Worker Assets deployment. Matching runs in the browser against the versioned public index and needs no model, D1 database, R2 bucket, secret, or runtime database connection.
