# California School Explorer v0.2.0

This release adds the first historical comparison year to the real-data website.

## Highlights

- Seven indicators now show 2023–24 and 2024–25 on the same timeline.
- The public bundle contains 1,839,368 school and district observations.
- The canonical PostgreSQL store contains 2,524,988 metric facts from 10 outcome snapshots plus one current school-profile snapshot.
- School-level two-year overlap ranges from 97.93% to 99.59% of current-year coverage, depending on the metric.
- Historical code `GZ` is preserved as the CDE-defined `Missing gender` subgroup rather than dropped or reassigned.

## Indicators

- ELA distance from standard
- Mathematics distance from standard
- Chronic absenteeism
- Suspension
- Four-year graduation
- A–G completion
- Four-year dropout

A–G completion is not the broader California Dashboard College/Career Indicator. That indicator remains a separate roadmap item.

## Reliability and comparability

All source files are pinned by URL, byte size, row count, header, encoding, and SHA-256 digest. Suppressed measures remain null, reported small samples are flagged, and no missing value is inferred. The adjacent years use the same project adapters and metric definitions. CDE nevertheless cautions that processing or reporting changes can affect comparisons across years.

See the [historical data quality report](../data-quality/2023-24-history.md) for import reconciliation and validation evidence.
